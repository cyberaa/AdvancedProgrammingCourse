Problem O - Roadside trees

Description

After a new road was build, many trees were planted at regular intervals along the road. Since young trees often die shortly after being planted, they were planted quite close together. As expected, many trees did die shortly after, but a good number of them survived.

As the trees grew bigger and bigger, it became clear that some were too close together. In addition, cutting some trees down would allow money to be raised by selling the timber. Therefore, the road department decided to leave standing only a number of trees, to be selected so that they were as far apart from each other as possible.

Would you have been able to help them?

Input

There may be several test cases in each input file. The first line of each test case consists of two integers: the original number of trees, n, and the number of trees to leave standing, k. Then, n lines follow, each containing the integer position of a tree along the road, in ascending order. A new test case may follow in the next line.

Output

For each test case, print the maximum distance that can be achieved between the two closest trees among the k trees left standing along the road.

Example

Example input:

16 7
21
258
289
304
321
351
427
448

537

596

Example output:
